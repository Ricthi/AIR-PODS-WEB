# 🎧 Airpods Animation Showcase

A stunning, interactive product landing page for Airpods with premium animations and dynamic themes.

---

## 🚀 Key Features

- **Interactive Carousel Slider:** Smooth, high-performance transitions between various Airpod models.
- **Dynamic Theme Engine:** Background and accent colors update automatically to match the selected product (Crimson, Midnight, Gold, etc.) for a cohesive brand experience.
- **Deep-Dive Specifications:** Click "SEE MORE" to reveal comprehensive details:
  - 🔋 **Used Time:** 6 hours
  - 🔌 **Charging Port:** Type-C
  - 📱 **Compatibility:** Android & iOS
  - 📶 **Bluetooth:** 5.3
  - 👆 **Controlled:** Touch
- **Modern Design System:** Utilizes glassmorphism, elegant typography (Inter/Roboto inspired), and fluid micro-animations for a premium look and feel.
- **Fully Responsive:** Perfectly optimized for seamless browsing on desktops, tablets, and smartphones.

## 🛠️ Built With

- **HTML5:** Semantic markup for structure and accessibility.
- **CSS3:** Advanced animations, custom variables, and flexbox/grid layouts.
- **JavaScript (ES6+):** Lightweight, vanilla JS for slider logic, event handling, and theme switching.

## 📂 Project Structure

```text
├── images/          # Product assets and visual elements
├── index.html       # Main landing page structure
├── style.css        # Core styling and animation logic
└── script.js       # Interactive functionality
```

1. **Clone the repository:**
